# Servercheck

The app can execute some Windows (network-test) commands.  
  
Currently:  
* ping  
* tracerout (tracert)  
  
#### App-start  
**servercheck.exe** 64 bit Windows  
**servercheck32.exe** 32 bit Windows  
**servercheckA32.exe** 32 bit ANSI Windows 
  
#### Download  
* From time to time there are some false positiv virus detections
[Virusscan](#virusscan) at Virustotal, see below.  

from github, these files are essential (rightclick, save as ...):

[servercheckA32.exe 32bit ANSI](https://github.com/jvr-ks/servercheck/raw/master/servercheckA32.exe)
